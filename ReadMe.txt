To run the application just run test.py

Follow along with the application, enter input parameters as asked by the app.

1.There's only one reaource i.e. a text file(test.txt) for current version of application

3.User will enter valid input as asked by the application not other than than.
4.test.py and test.txt files must be in same directory.
5.Initially there are two users in the app : Admin and User1.
6.Admin has all sorts of permissions whereas User1 has ReadOnlyAccess.
7.Write means you want to overwrite the file. Append means you will append the file.
